# Security Policy

## Supported Versions

The following versions of BinaryScouts components are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| v2.x    | Yes       |
| v1.x    | No        |

## Reporting a Vulnerability

We take the security of our services seriously. If you find a security vulnerability, please do not open a public GitHub issue. Instead, report it directly to our security response team:

1. **Email**: Send detailed vulnerability reports to **security@binaryscouts.com**.
2. **Details**: Please include a clear description of the issue, step-by-step reproduction instructions, and any proof-of-concept scripts or commands.
3. **Response**: We will acknowledge receipt of your report within 48 hours and work with you to patch and coordinate release timelines.
